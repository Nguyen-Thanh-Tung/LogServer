# LogServer
Log Server for DSD06
# Software required
- Webstorm : IDE to code
- Prepros: Software for compile sass/scss => css
- Robomongo: View database NoSQL
- Postman: Test GET/POST/PUT/DELETE

# Language
- NodeJs (Server)
- MongoDB (Database)
- Jade (View)

# Init, start server
Step 1: Import module

    npm install

Step 2: Start server
    
    npm start
